Step by Step Instructions

  1. Create a fresh profile/instance of Minecraft. If using the default launcher, it is recommended to create a different installation directory for the profile you will be using for Foli Client. I strongly suggest using MultiMC, as it makes it easy to manage multiple Minecraft profiles, and allows you to install Fabric with one click.
  2. Install Fabric. Do not install Fabric API, as the appropriate version is included in the modpack.
  3. Go to the link above to find the latest release, and download the installer for your OS.
  4. On Windows or MacOS, run the installer for your OS (FoliClientInstaller-x.x.x-Win.exe for Windows, FoliClientInstaller-x.x.x-Mac for Mac). On Linux, download the source code archive and run the python script. When prompted, paste in the full filepath of your Minecraft install directory (e.g. C:\Users\ABlazingEBoy\Documents\MinecraftDirectory on Windows, /Users/ABlazingEBoy/Documents/MinecraftDirectory on Mac, or /home/ABlazingEBoy/Documents/MinecraftDirectory on Linux). The installer is pretty self-explanatory.
  5. After you start up Foli Client, make sure to apply the two included resource packs. Foli Client Specific Textures should always be on top of your resourcepack list for the client to look correct.
  6. Enjoy using Foli Client! After installing, I suggest going into control settings and editing the key bindings to your liking, as well as going into video settings and changing those to suit your hardware. If you're having any difficulties, feel free to contact me at ablazingeboy#7375 on Discord.

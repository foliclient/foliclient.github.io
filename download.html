Step by Step Instructions

  1. Create a fresh profile/instance of Minecraft. If using the default launcher, it is recommended to create a different installation directory for the profile you will be using for Foli Client. I strongly suggest using MultiMC, as it makes it easy to manage multiple Minecraft profiles, and allows you to install Fabric with one click.
  2. Install Fabric. Do not install Fabric API, as the appropriate version is included in the modpack.
  3. Download the latest release from the link above. If on Windows, download the .exe file. If on MacOS or Linux, download the 'source code' zip and unzip it. Do not move any of the files from the folder where they were unzipped (Note that if you are on MacOS or Linux you will need to install python 3 to run the installer).
  4. Run the installer for your OS (FoliClientInstaller.exe for Windows, and FoliClientInstaller.py for other platforms). If on MacOS or Linux, you might need to open a terminal in the source code folder and run python3 FoliClientInstaller.py. When prompted, paste in the full filepath of your Minecraft install directory (e.g. C:\Users\ABlazingEBoy\Documents\MinecraftDirectory on Windows or /Users/ABlazingEBoy/Documents/MinecraftDirectory on Mac). The installer is pretty self-explanatory.
  5. After you start up Foli Client, make sure to apply the two included resource packs. Foli Client Specific Textures should always be on top of your resourcepack list for the client to look correct.
  6. Enjoy using Foli Client! I suggest going into control settings and editing the key bindings to your liking. If you're having any difficulties, feel free to contact me at ablazingeboy#7375 on Discord.
